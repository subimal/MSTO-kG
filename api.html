MSTOOptpar.AggressiveRefinement(mol, fname, trials=10, tol=1e-16, fci=True, itermax=100)¶

Performs a local, gradient-free iterative search (hill-climbing) to fine-tune basis set parameters. This exploitation phase provides the precision necessary for scientific accuracy.

Parameters:

  • mol – Best molecule object from the global search phase.

  • fname – Path to save the refined candidate.

  • trials – Number of independent refinement trials.

  • tol – Energy tolerance for local convergence.

  • fci – Whether to include FCI results in the saved output.

  • itermax – Max iterations permitted per refinement trial.

Example:

>>> mol_best = loadBSI("MOBS_Li_msto-03g.pickle")['mol_CISD'].mol
>>> AggressiveRefinement(mol_best, "MOBS_Li_refined.pickle", trials=5)
MSTOOptpar.BasisReshape(mol, k)¶

Reshapes the STO-nG basis of the molecule to cardinal number k.

Parameters:

  • mol (pyscf.gto.mole.Mole) – the pyscf molecule object

  • k (int) – Cardinal number for which the basis is to be reshaped.

Returns:

the pyscf molecule object with reshaped basis

Return type:

pyscf.gto.mole.Mole

Example:

>>> element = 'H' # choose your element
>>> mol = moldict('sto-6g')[element] # get the default molecule in your basis
>>> mol.basis
{'H': [[0,
   [35.52322122, 0.00916359628],
   [6.513143725, 0.04936149294],
   [1.822142904, 0.1685383049],
   [0.625955266, 0.3705627997],
   [0.243076747, 0.4164915298],
   [0.100112428, 0.1303340841]]]}
>>> mol1 = BasisReshape(mol, 7)
>>> mol1.basis # The new coefficients are appended as [1.0, 1.0] for the deficit cardinal numbers
{'H': [[0,
   array([3.55232212e+01, 9.16359628e-03]),
   array([6.51314372, 0.04936149]),
   array([1.8221429, 0.1685383]),
   array([0.62595527, 0.3705628 ]),
   array([0.24307675, 0.41649153]),
   array([0.10011243, 0.13033408]),
   array([1., 1.])]]}
>>> mol2 = BasisReshape(mol1, 5)
>>> mol2.basis # The excess coefficients are removed.
{'H': [[0,
   array([3.55232212e+01, 9.16359628e-03]),
   array([6.51314372, 0.04936149]),
   array([1.8221429, 0.1685383]),
   array([0.62595527, 0.3705628 ]),
   array([0.24307675, 0.41649153])]]}

This molecule object can now be used to optimize the basis set or compute the energies.

MSTOOptpar.DumpStandardBasis(element, basis, dirname='./', forcedump=False)¶

Create the molecule object for a molecule in a given basis and dump in a file. A new file will be created only if it does not exist. The format of the filename will be BS_<element>_<basis>.pickle and stored in the specified directory.

Parameters:

  • element (str) – A molecule defined in MSTOOpt.moldict().

  • basis (str) – A basis name defined in basis_set_exchange package for which the energy is to be computed.

  • dirname (str) – The directory where the molecule object is to be dumped.

Example:

>>> DumpStandardBasis('H', 'sto-4g', dirname='./')
MSTOOptpar.FetchBestMol(element, k, dirname='./')¶

Fetches the best available basis for optimization of element for cardinal number k.

The order is as follows for a cardinal number k:

  • MSTO-kG

  • STO-kG

  • MSTO-(k-1)G

  • STO-(k-1)G

It is best to ensure that at least the STO-2G basis is available.

MSTOOptpar.LogicalRefinement(element, Nmax, trials=20, tol=1e-16, trials_mu=15, trials_co=20, prefix='MOBS', dirname='./', fci=True, itermax=25, agg_trial=True, ga_trial=True, kmax=12)¶

Refine the MSTO basis sets generated. For 3 <= k <= 11 if the FCI energy for a given k is larger than that for k-1, an optimization is performed.

  • Nmax (int) – Population of candidates to consider in the optimization

  • trials (str) – The number of generations to run

  • tol (float) – Tolerance of energy for convergence

  • tol_mu – Number of mutations to consider under each trial

  • trials_co (int) – Number of crossovers to consider after each mutation

  • prefix (str (optional. Defaults to MOBS)) – Prefix of the pickled database

  • dirname (str (optional. Defaults to ./)) – Directory to look for the dumped database files.

Returns:

Nothing

Return type:

None

Example:

>>> element = 'H'
>>> Nmax = 10
>>> LogicalRefinement(element, Nmax, trials=20, tol=1e-16, trials_mu=15, trials_co=20, prefix = 'MOBS', dirname=dirname, klst=range(3, 12))

MSTOOptpar.OptimizeElement(mol, Nmax, fname, trials=10, tol=1e-10, trials_mu=100, trials_co=100)¶

Optimizes basis set coefficients using a hybrid Memetic Algorithm. Evolves a population of basis sets toward a lower RCISD ground state energy.

Parameters:

  • mol (pyscf.gto.mole.Mole) – Initial PySCF molecule object used as a template.

  • Nmax (int) – Population size of candidates for global exploration.

  • fname (str) – Path to the pickle file where the best candidate is updated.

  • trials (int) – Number of major optimization cycles.

  • tol (float) – Energy tolerance for population convergence (Hartree).

  • trials_mu (int) – Number of mutation attempts per major cycle.

  • trials_co (int) – Number of crossover operations per mutation trial.

Returns:

Tuple of (final population, generation history, energy history).

Return type:

tuple

Example:

>>> element = 'Li'
>>> mol = moldict('sto-3g')[element]
>>> fname = "MOBS_Li_msto-03g.pickle"
>>> pop, gen, en = OptimizeElement(mol, 5, fname, trials=2)

MSTOOptpar.dumpBSI(BSI_instance, fname='BS_EO_pickled')¶

An instance of the computed basis sets can be dumped as a pickle file. It will be useful if one wishes to reuse these values for further improvement.

  • fname (str) – The filename where the pickled object will be dumped.

Returns:

Nothing

Return type:

None

MSTOOptpar.getEnergies(mol, nroots=4, fci=False, cisd=True)¶

Retrieve the Hartree-Fock, FCI, and CISD energies for a molecule. The method ensures the molecule object is built locally to maintain consistency in parallel environments.

Parameters:

  • mol (pyscf.gto.mole.Mole) – The molecule whose energies are to be determined.

  • nroots (int, defaults to 4.) – Number of roots for the FCI solver.

  • fci (bool, defaults to False.) – Whether to compute Full Configuration Interaction energy.

  • cisd (bool, defaults to True.) – Whether to compute Configuration Interaction Singles and Doubles energy.

Returns:

Dictionary containing ‘HF’, ‘FCI’, and ‘CISD’ energies.

Return type:

dict

Example:

>>> mol = moldict('sto-3g')['H2']
>>> energies = getEnergies(mol, cisd=True)
>>> print(energies['CISD'])
-1.11671432537233

MSTOOptpar.putcoeffs(iden, b)¶

Creates a new basis set for a pyscf.gto.mole.Mole object from the element/shell identifiers and the array of basis coefficients. These need not be created by hand (you can if you want to and know what you are doing). This function was created to modify the basis of the molecule object and then inject the modified basis back into the molecule. The best way to use it would be to extract the identifier and basis set array using MSTOkG.MSTOOpt.getcoeffs(), modify the basis appropriately and then use this function to generate the new basis. This new basis should be used to replace the basis in the pyscf.gto.mole.Mole object (and subsequently build it).

Parameters:

  • iden (list) – A list of element name and shell identifiers

  • b (numpy.array) – Basis set coefficients.

Returns:

Basis set for pyscf.gto.mole.Mole object

Return type:

numpy.ndarray

Example:

>>> element = 'LiH' # choose your element
>>> mol = moldict('sto-2g')[element] # retrieve the molecule with standard parameters in the sto-2g basis
>>> iden, basis = getcoeffs(mol) # extract the identifiers and the basis set coefficients
>>> iden
[['H', 0], ['Li', 0], ['Li', 0], ['Li', 1]]
>>> basis
array([[[1.30975638, 0.4301285 ],
    [0.23313597, 0.67891353]],
   [[6.16384503, 0.4301285 ],
    [1.09716131, 0.67891353]],
   [[0.24591632, 0.04947177],
    [0.06237087, 0.96378241]],
   [[0.24591632, 0.51154071],
    [0.06237087, 0.6128199 ]]])
>>> basis[0][1][0] # changing this element to 0.222222
np.float64(0.2331359749)
>>> new_basis = basis[:] # creating a copy
>>> new_basis[0][1][0] = 0.222222 # modifying the copy
>>> new_basis # let's check
array([[[1.30975638, 0.4301285 ],
    [0.222222  , 0.67891353]],
   [[6.16384503, 0.4301285 ],
    [1.09716131, 0.67891353]],
   [[0.24591632, 0.04947177],
    [0.06237087, 0.96378241]],
   [[0.24591632, 0.51154071],
    [0.06237087, 0.6128199 ]]])
>>> new_basis_set = putcoeffs(iden, basis) # create the basis from the new coefficients
>>> from copy import deepcopy # we need this to create a copy of the molecule
>>> mol1 = deepcopy(mol) # `mol1` is the new copy
>>> mol1.basis = new_basis # modify the basis of the new molecule
>>> mol1.basis # now build this molecule and proceed with your computations
array([[[1.30975638, 0.4301285 ],
    [0.222222  , 0.67891353]],
   [[6.16384503, 0.4301285 ],
    [1.09716131, 0.67891353]],
   [[0.24591632, 0.04947177],
    [0.06237087, 0.96378241]],
   [[0.24591632, 0.51154071],
    [0.06237087, 0.6128199 ]]])
